# 바이브코딩 요청 가이드

사장님은 코딩을 몰라도 됩니다. 아래처럼 말하면 됩니다.

## 새 프로젝트 시작

```text
웹사이트 하나 만들자. 목적은 [목적]이고, 필요한 화면은 [화면 목록]이야.
초보자 기준으로 설명하면서 만들어줘.
```

## 기능 추가

```text
현재 프로젝트에 [기능]을 추가해줘.
완료 후 내가 어떻게 실행해서 확인하는지도 알려줘.
```

## 오류 해결

```text
이 오류가 나왔어.
[오류 메시지 붙여넣기]
원인과 해결 방법을 쉽게 설명하고 고쳐줘.
```

## 코드 설명

```text
이 파일이 무슨 역할인지 초보자도 이해하게 설명해줘.
```

## 좋은 요청에 들어가면 좋은 정보

- 만들고 싶은 결과
- 참고 사이트나 이미지
- 꼭 필요한 기능
- 싫어하는 방식
- 마감이나 우선순위
